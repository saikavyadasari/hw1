About This Website

This website is a simple, static website built using HTML and CSS. It consists of three pages: index.html, about.html, and blog.html. The website follows a basic structure with a header, main content area, and footer.

Tech Stack:

  1. HTML (Hypertext Markup Language): HTML is used to structure the content of the web pages. It provides the basic elements such as headings, paragraphs, links, and lists.
  2. CSS (Cascading Style Sheets): CSS is used to style and visually enhance the HTML elements. In this website, CSS is used to set the background color, font styles, layout, and styling of the navigation menu.

How it Works:

  1. Structure: The website follows a standard HTML structure with the <!DOCTYPE html>, <html>, <head>, and <body> tags. The <head> section contains metadata like the page title and CSS styles, while the <body> section contains the visible content of the web page.
  2. Navigation: The navigation menu is created using an unordered list (<ul>) inside the <header> section. Each navigation link is represented by a list item (<li>) with an anchor tag (<a>) pointing to the respective HTML file. The CSS styles are applied to the navigation menu to make it horizontal, remove default list styles, and add hover effects.
  3. Content: The main content of each page is contained within the <main> section. The index.html file contains a welcome message, the about.html file has information about the website and the author, and the blog.html file is meant to showcase an excerpt from a book or article.
  4. Styling: The CSS styles are embedded within the <head> section of each HTML file using the <style> tag. The styles define the overall look and feel of the website, including the dark background color, white text color, font styles, and layout.
  5. Active Page Highlight: The navigation menu includes a visual cue to indicate the currently active page. This is achieved by adding the class="active" attribute to the corresponding navigation link. The CSS styles target the .active class and apply the same background color as the hover effect, making the active link stand out.
  6. Linking: The navigation links connect the different pages of the website. When a user clicks on a link, the corresponding HTML file is loaded and displayed in the browser. The relative paths (../index.html, about.html, blog.html) are used to navigate between the pages.

Overall, this website showcases a basic understanding of HTML and CSS, including structure, navigation, content organization, and styling. It demonstrates the fundamental concepts of building a multi-page website with a consistent look and feel across all pages.

While this website is static and does not involve any server-side processing or dynamic content, it serves as a starting point for learning web development and can be further extended with additional features and functionality as needed.
